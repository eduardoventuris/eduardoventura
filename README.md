# Mapa Curriculos

Versao do sistema de gestao de curriculos com identidade visual em estilo estrutura de mural funcional.

## Resumo

- Composicao visual inspirada em quadro de acompanhamento.
- Persistencia local, validacao e pagina dinamica mantidas.
- Projeto academico construido com Next.js App Router e Tailwind.

## Rotas principais

- `/`
- `/sistema/paginas/curriculos`
- `/sistema/paginas/curriculos/[id]`
- `/sistema/paginas/curriculos/novo`

## Execucao

```bash
npm install
npm run dev
```
